A creative WordPress blog/magazine theme

== Description ==

Milk-It is a creative WordPress blog/magazine theme built for blogs, newspapers, magazines or review sites. Developed with SEO best practices in mind (and microdata optimized), Milk-It lets you to write articles and blog posts easily and quickly.

== Installation ==

Upload the **milkit.zip** file in 'Appearance > Themes > Add New > Upload' or upload via FTP the extracted **milkit** folder in 'wp-content/themes/'.
